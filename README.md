# System Design Hub

A modern, clean, and responsive React application for exploring system design articles.

## Features
- **Modern UI**: Clean, Medium-inspired minimalist design.
- **Dark Mode**: Toggle between light and dark themes.
- **Search & Filter**: Real-time search and tag-based filtering.
- **PDF Viewer**: Embedded PDF viewing for detailed system design guides.

## Tech Stack
- React + Vite
- React Router
- Lucide Icons
- Vanilla CSS with CSS Variables

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
